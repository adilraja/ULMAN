Description

Demonstrates connecting to a SSH server and performing a filtered directory listing using the command
line "filter" argument passed in to the run.bat or run.sh file.

Instructions

From the shell prompt enter the following commands based on your operating system.  
Update the variables in the sshdirlistingfiltered.txt file to match your server environment.
Update the filter variable in the run.bat and run.sh files to meet your needs.

Unix

chmod a+x run.sh
run.sh

Windows (DOS)

run.bat
